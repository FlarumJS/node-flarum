# node-flarum

*Express Middleware*
A forum made out of [Flarum](http://flarum.org)'s design forum with NodeJS

## Installation

	npm install node-flarum --save


## Usage

	var flarum = require('flarum');

	var express = require('express');
	var app = express();

	// Other stuff

	app.use('/forum', flarum)


## Tests

	npm test
